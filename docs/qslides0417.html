통계학이란?

통계학이란 답이없는 세상에서 답을 내려주는 학문이라 생각합니다. (최규빈)

- 정답을 연구하는 학문 vs 해법을 연구하는 학문

  • https://www.youtube.com/shorts/cxNfODpuPE0

- 통계학은 해법을 연구하는 학문에 가까움

- 통계학은 자연과학이지만 공학적인 느낌도 가지고 있음.

Example 1

- 대수학교육에 대한 리처드파인만에 생각

- 핵심아이디어

  • 질문: \(2x+7= 9\) 를 푸는방법?
  • 방법1: 양변에 7을 빼세요, 그럼 \(2x=2\) 가 나오죠? 우리가 구하고 싶은건 \(2x\)가 아니라 \(x\) 잖아요? 그런데 \(x\)에 2가 곱해져있으니까 양변을 2로 나누면 되겠죠? 이것이 일차방정식을 풀어내는 방법입니다.
  • 방법2: 1을 넣으니까 식이 성립하네? \(\to\) 답이네? (요게 통계학과 느낌)
